# **App Name**: FactCheck AI

## Core Features:

- Text Selection & Verification Trigger: Users can select text on any webpage and initiate the factual accuracy check via a context menu option or a popup button in the browser.
- AI Factual Verification Tool: Utilizes a generative AI model to analyze selected text for factual accuracy by comparing it against reliable sources and identifying potential misinformation.
- Clear Verdict Display: Presents a concise verdict on the text's accuracy (e.g., 'Likely Accurate,' 'Needs Verification,' 'Potentially Misleading') in an easily digestible format.
- Correction & Context Tool: When inaccuracies are identified, the AI tool suggests brief corrections or provides additional context to clarify the information.
- Popup Interface: Provides a clean and intuitive popup interface for settings, previous checks (if applicable in future versions, for MVP just basic interaction), and displaying results.

## Style Guidelines:

- Primary color (for interactive elements and success): A vibrant yet authoritative green (#36B836) that signifies reliability and truth. This green offers clarity and pairs well with a light background.
- Background color: A very light, almost white, subtle green-tinted shade (#EEF5EE) providing a clean and non-distracting canvas. It shares the hue of the primary color but is heavily desaturated and bright.
- Accent color (for warnings and critical information): A clear and distinct red (#E03D3D) to highlight potential misinformation or items needing verification, ensuring immediate attention without being overly aggressive.
- Body and headline font: 'Inter', a modern, objective sans-serif, offering excellent legibility and a neutral, professional feel suitable for an informative application.
- Use minimalist line icons. Checkmarks for accurate, question marks or exclamation points for verification needed, and cross icons for misleading information. Icons should be clear and immediately understandable.
- Clean and organized layout within the browser extension popup, using a clear hierarchy for results. Ample white space to enhance readability and a focused display of the verdict and suggestions.
- Subtle loading animations (e.g., a simple spinner) while the AI processes the text, indicating active verification without distracting the user. Smooth transitions for verdict display.
