# FactCheck AI: Deep Forest Clarity

FactCheck AI is a high-integrity factual verification platform designed to navigate the "fog of misinformation" using advanced Generative AI and a grounded, atmospheric design language.

## The Core Concept
The project is built on the philosophy of **"Truth through Atmosphere."** Unlike clinical or generic AI tools, FactCheck AI utilizes a "Twilight Forest" aesthetic (15% darker than standard UIs) to create a focused, high-contrast environment for critical thinking.

## Key Features

### 1. Multi-Modal Verification (Tier 1)
- **Text Analysis**: Instant auditing of pasted text or selected snippets.
- **Image Intelligence**: Extracting and verifying claims found in visual artifacts like tweets, news headlines, and memes.
- **Authority Sources**: Every verdict is backed by high-authority citations (NASA, BBC, Wikipedia, etc.) with AI-assessed reliability scores.

### 2. 2nd Tier Deep Scan (Tier 2)
- **Semantic Nuance**: Goes beyond "True/False" to identify gray areas and subtle linguistic misconceptions.
- **Cross-Reference Engine**: Provides a deeper study layer with 2-3 additional authoritative sources and a final "Chain Confidence" score.

### 3. Seamless Extension Demo
- **Highlight-to-Verify**: A simulated browser experience showing how a chrome extension would provide instant, floating results near selected text without page blurs or interruptions.

### 4. Intelligence Archive
- **Persistent History**: All verifications are securely stored in a Firebase-backed timeline, allowing users to build a personal vault of factual intelligence.

### 5. Authoritative TTS
- **Narration**: One-click professional narration of fact-checking reports using high-quality Text-to-Speech models.

## Design System
- **Theme**: Twilight Forest (Primary: #051a14, Background: #f4f7f6).
- **Components**: "Misty Glass" with 15% deeper shadows for weighted depth.
- **Transitions**: Retro "TV Shutoff" CRT-style intro for a grounded tech feel.
- **Inputs**: Spacious `px-12` padding with a clean border, shifting placeholder text for a premium, intentional layout.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI Orchestration**: Genkit 1.x (Gemini 3.1 Flash Lite)
- **Backend**: Firebase (Auth & Firestore)
- **Styling**: Tailwind CSS + ShadCN UI
